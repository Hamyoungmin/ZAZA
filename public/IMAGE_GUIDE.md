# 제주도 여행 웹사이트 - 이미지 가이드

## 필요한 이미지 파일들

이 프로젝트에서는 다음 이미지 파일들이 필요합니다. 실제 이미지로 교체해주세요.

### 메인 히어로 캐러셀 이미지
- **Main.png** - 첫 번째 슬라이드 배경 (제주 해안 사진) - 이미 있음 ✓
- **Main(Right).png** - 두 번째 슬라이드 배경 (제주 풍경 사진) - 이미 있음 ✓

권장 크기: 1920x600px 이상

### 추천 여행지 카드 이미지
- **travel1.jpg** - 열기구 사진
- **travel2.jpg** - 공항/여행 사진
- **travel3.jpg** - 산/호수 풍경 사진

권장 크기: 800x600px

### 원형 섹션 이미지
- **circle1.jpg** - 인테리어 사진
- **circle2.jpg** - 산 풍경 사진
- **circle3.jpg** - 추상적인 색상 사진

권장 크기: 500x500px (정사각형)

## 이미지 교체 방법

1. `public/` 폴더에 있는 placeholder 파일들을 삭제
2. 실제 이미지를 같은 이름으로 `public/` 폴더에 복사
3. 또는 컴포넌트 파일에서 파일명을 수정

## 이미지 최적화 팁

- Next.js의 `Image` 컴포넌트 사용 권장
- WebP 포맷 사용으로 용량 절감
- 적절한 해상도로 리사이징하여 로딩 속도 개선
